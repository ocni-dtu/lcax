LCAx's calculation engine allows you to calculate environmental impacts for different levels of a project.

The calculation functions take an object (Project, Assembly, or Product) and a `CalculationOptions` object.

#### `CalculationOptions`

**referenceStudyPeriod:** `number | null` \
**lifeCycleModules:** `LifeCycleModule[]` \
**impactCategories:** `ImpactCategoryKey[]` \
**overwriteExistingResults:** `boolean`

The fields are:

- `referenceStudyPeriod`: The period over which the impacts are calculated. If null, the project's reference study period is used.
- `lifeCycleModules`: A list of life cycle modules to include in the calculation (e.g., A1-A3, A4, B4, C3, C4).
- `impactCategories`: A list of impact categories to calculate (e.g., GWP, ODP, POCP).
- `overwriteExistingResults`: If true, existing results will be overwritten.
