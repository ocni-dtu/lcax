LCAx is available in several coding languages.
In the following you can see how it can be installed in each supported language.

## JavaScript

```bash
npm install lcax
```

## Python

```bash
pip install lcax
```

## Rust

```bash
cargo add lcax
```

## C#

```bash
nuget add lcax
```
